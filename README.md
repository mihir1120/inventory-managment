# Ashokvatika Android App

Jetpack Compose implementation of the uploaded mobile login design.

## Implemented UI behavior

- Username field with visibility toggle.
- Password field with visibility toggle.
- Sign in validation and simulated success state.
- Clear button resets the form.
- Forgot password dialog responds to the entered username.

## Project note

This workspace did not include Java, Gradle, Android SDK, or ADB, so the code could not be built or run here. Once an Android toolchain is installed, open the folder in Android Studio and run the `app` module.
